Matemáticas Financieras

Capítulo 2 Anualidades

Considere la siguiente transacción:

Se deposita \(\$1\) cada periodo durante \(n\) periodos, i.e, hay \(n\) depósitos de \(\$1\).

  1. Supongamos interés compuesto, ¿cuánto dinero acumulado se tiene justo después del último depósito?
  • Justo después del \(2\text{°}\) depósito, ¿cuánto dinero acumulado hay?

\(1(1+i)^1+\underbrace{1}_{\scriptstyle \text{peso adicional}}\)

  • Justo después del \(3\text{°}\) depósito, ¿cuánto dinero tenemos?

\(\underbrace{\left[1(1+i)^1 + 1 \right]}_{\scriptstyle \text{Ya lo teníamos}}(1+i) + \underbrace{1}_{\scriptstyle \text{peso adicional}} = (1+i)^2 +(1+i) +1\)

  • Justo después del \(4\text{°}\) depósito, ¿cuánto?

\(\left[(1+i)^2 + (1+i) +1 \right](1+i) +1 = (1+i)^3 +(1+i)^2 + (1+i) +1\)

Inductivamente, justo después del \(n\text{-ésimo}\) depósito, ¿cuánto dinero tenemos? \[=\underbrace{(1+i)^{n-1}+(1+i)^{n-2}+\ldots+(1+i) +1}_{\scriptstyle n\, \text{sumandos}} = X\]

  • ¿Cuánto dinero hay un periodo después del último depósito?

\[\begin{align*} &X(1+i) = Y\\ Y &= (1+i)\left[(1+i)^{n-1} + (1+i)^{n-2}+ \ldots + (1+i) +1 \right] \\ &= \underbrace{(1+i)^n + (1+i)^{n-1} + \ldots + (1+i)}_{\scriptstyle n\, \text{sumandos}} \end{align*}\]

  • Si en vez de esa maroma de hacer \(n\) depósitos y simplemente hacer un único depósito en la misma fecha del \(1\text{°}\) pago y obtener el mismo valor acumulado.

¿De cuánto tendría que ser este depósito?

\[\begin{align*} &W\,\text{debe satisfacer:} \: W(1+i)^n = Y\\ &\text{ó bien} \, W(1+i)^{n-1} = X \: \Longrightarrow \: W=Y(1+i)^{-n}\\ & W = \left[(1+i)^n + (1+i)^{n-1}+\ldots + (1+i) \right] (1+i)^{-n}\\ &= \underbrace{1+ (1+i)^{-1} + \ldots + (1+i)^{-(n-1)}}_{\scriptstyle n\, \text{sumandos}} \end{align*}\]

  • Si en vez de esa maroma de hacer \(n\) depósitos y simplemente hacer un único depósito un periodo antes de la fecha del \(1\text{°}\) pago.

¿De cuánto debe ser dicho depósito?

\[\begin{align*} &Z\,\text{debe satisfacer:}\: Z(1+i)^n=X \: \text{ó bien} \: Z(1+i)^{n+1} = Y\\ & \Longrightarrow \: Z= X(1+i)^{-n} = \left[(1+i)^{n-1} + (1+i)^{n-2} + \ldots + (1+i) + 1 \right](1+i)^{-n}\\ &\hspace{1.5cm} = (1+i)^{-1} +(1+i)^{-2} +\ldots +(1+i)^{-(n-1)} + (1+i)^{-n} = (*) \end{align*}\]

Notación

\((*)=\underbrace{V+V^2+\ldots + V^{n-1}+V^n}_{\scriptstyle n\, \text{sumandos}}\)

En resumen

  1. \(X=(1+i)^{n-1}+(1+i)^{n-2}+\ldots+(1+i) +1 \: :\) \(\require{enclose}s_{\enclose{actuarial}{n}i}\)

  2. \(Y = (1+i)^n + (1+i)^{n-1}+\ldots+(1+i) \: \: :\) \(\require{enclose}\ddot{s}_{\enclose{actuarial}{n}i}\)

  3. \(W = 1 + V + V^2 + \ldots + V^{n-1} \: :\) \(\require{enclose}\ddot{a}_{\enclose{actuarial}{n}i}\)

  4. \(Z = V+V^2+\ldots +V^n \: :\) \(\require{enclose}a_{\enclose{actuarial}{n}i}\)

\(i \: \longrightarrow \: \text{tasa efectiva de interés por periodo}\\ n \: \longrightarrow \: \text{número de pagos}\)

\(\require{enclose}s_{\enclose{actuarial}{n}i}, \; \require{enclose}\ddot{s}_{\enclose{actuarial}{n}i}, \; \require{enclose}\ddot{a}_{\enclose{actuarial}{n}i}, \;\require{enclose}a_{\enclose{actuarial}{n}i}, \; ; n \in \mathbb{N}\)

Lema 2.1 Para \(x\in \mathbb{R}, \, k, \, n \in \mathbb{N}_+, \, k<n\). Sea \(S= x^k + x^{k+1} + \ldots + x^n\), entonces \[\boxed{S = \frac{x^k-x^{n+1}}{1-x}}\] \(\cdot\) Las fórmulas de las anualidades se deducen de este lema.

Demostración. \[\begin{align*} & S = x^k + x^{k+1} + \ldots + x^n \\ & xS = \hspace{0.5cm} x^{k+1} + \ldots + x^n + x^{n+1} \\ & \Longrightarrow S-xS = x^k -x^{n+1} \\ & \Longrightarrow S = \frac{x^k-x^{n+1}}{1-x} \end{align*}\]

Proposición 2.1 .

  1. Si \(i \neq 0\), entonces \(\boxed{\require{enclose}a_{\enclose{actuarial}{n}i} = \frac{1-V^n}{i}} = \frac{1-(1+i)^{-n}}{i}\)

  2. Si \(i \neq 0\), entonces \(\boxed{\require{enclose}\ddot{a}_{\enclose{actuarial}{n}i} = \frac{1-V^n}{d}} = \frac{1-(1+i)^{-n}}{d}\), donde \(d \sim i\)

  3. Si \(i \neq 0\), entonces \(\boxed{\require{enclose}s_{\enclose{actuarial}{n}i} = \frac{(1+i)^n -1}{i}}\)

  4. Si \(i \neq 0\), entonces \(\boxed{\require{enclose}\ddot{s}_{\enclose{actuarial}{n}i} = \frac{(1+i)^n -1}{d}}\), con \(d\sim i\)

  5. Si \(i = 0\), entonces \(\require{enclose}a_{\enclose{actuarial}{n}i} = \require{enclose}\ddot{a}_{\enclose{actuarial}{n}i} = \require{enclose}s_{\enclose{actuarial}{n}i} = \require{enclose}\ddot{s}_{\enclose{actuarial}{n}i}\)

\[\begin{align*} \require{enclose} r(Ga)_{\enclose{actuarial}{n}i} &= \underbrace{1V + (1+r)V^2 + (1+r)^2V^3 + \ldots + (1+r)^{n-2}V^{n-1} + (1+r)^{n-1}V^n}_{\scriptstyle n \, \text{sumandos}} \\ &= V\left[ 1+(1+r)V + [(1+r)V]^2 + \ldots + [(1+r)V]^{n-2} + [(1+r)V]^{n-1}\right]\\ & \mathrel{\stackrel{\scriptstyle \text{lema}}{=}} V\left[\frac{[(1+r)V]^0 - [(1+r)V]^{n-1+1}}{1-(1+r)V} \right] \: \longleftarrow \: x = (1+r)V\\ &= V\left[\frac{1-(1+r)^nV^n}{1-(1+r)V} \right] = \frac{1-(1+r)^nV^n}{V^{-1}[1-(1+r)V]} = \frac{1-(1+r)^nV^n}{V^{-1}-(1+r)} \\ & = \frac{1-(1+r)^n V^n}{(1+i)-(1+r)} = \boxed{\frac{1-\left(\frac{1+r}{1+i} \right)^n }{1-r}, \: i \neq r}\\ &\hfill\\ \require{enclose} r(G\ddot{s})_{\enclose{actuarial}{n}i} &\mathrel{\stackrel{\scriptstyle \text{Def}}{=}} (1+r)^{n-1}(1+i) + (1+r)^{n-2}(1+i)^2 + \ldots + (1+r)^2(1+i)^{n-2} \\ &\quad + (1+r)(1+i)^{n-1} + 1(1+i)^n \\ &= (1+r)^n \left[\frac{1+i}{1+r} + \left(\frac{1+i}{1+r} \right)^2 + \ldots + \left( \frac{1+i}{1+r} \right)^n \right] \\ &\mathrel{\stackrel{\scriptstyle \text{lema}}{=}} (1+r)^n \left[\frac{\left(\frac{1+i}{1+r} \right)^1 - \left( \frac{1+i}{1+r} \right)^{n+1} }{1 - \left(\frac{1+i}{1+r} \right) } \right] \: \longleftarrow \: x = \frac{1+i}{1+r}\\ & = \frac{(1+r)^n\left( \frac{1+i}{1+r}\right)\left[1 - \left(\frac{1+i}{1+r} \right)^n \right] }{\frac{1+r-1-i}{1+r}} = \frac{(1+i)(1+r)^{n-1}\left[1 - \frac{(1+i)^n}{(1+r)^n} \right] }{\frac{r-i}{1+r}}\\ &= \frac{(1+i)(1+r)^n\left[1 - \frac{(1+i)^n}{(1+r)^n} \right] }{r-i} = \frac{(1+i)\left[(1+r)^n - (1+i)^n \right] }{r-i}\\ & = \boxed{\frac{(1+i)^n - (1+r)^n}{\frac{i-r}{1+i}}} \end{align*}\]

Proposición 2.2 .

  1. Si \(i\neq r, \:\require{enclose} r(Ga)_{\enclose{actuarial}{n}i} = \frac{1- \left(\frac{1+r}{1+i} \right)^n }{i-r}\)

  2. Si \(i\neq r, \: \require{enclose} r(G\ddot{a})_{\enclose{actuarial}{n}i} = \frac{1- \left(\frac{1+r}{1+i} \right)^n}{\frac{i-r}{1+i}}\)

  3. Si \(i\neq r, \: \require{enclose} r(Gs)_{\enclose{actuarial}{n}i} = \frac{(1+i)^n - (1+r)^n}{i-r}\)

  4. Si \(i\neq r, \: \require{enclose} r(G\ddot{s})_{\enclose{actuarial}{n}i} = \frac{(1+i)^n - (1+r)^n}{\frac{i-r}{1+i}}\)

  5. Si \(i= r, \: \require{enclose} r(G\ddot{a})_{\enclose{actuarial}{n}i} = r(Ga)_{\enclose{actuarial}{n}i} = r(Gs)_{\enclose{actuarial}{n}i} = r(G\ddot{s})_{\enclose{actuarial}{n}i} = n\)

\[\begin{align*} \require{enclose} r(G\ddot{a})_{\enclose{actuarial}{n}i} &\mathrel{\stackrel{\scriptstyle \text{Def}}{=}} 1 + (1+r)V + (1+r)^2V^2 + \ldots + (1+r)^{n-1}V^{n-1}\\ &= 1 + \frac{1+r}{1+i} + \left(\frac{1+r}{1+i} \right)^2 + \ldots + \left( \frac{1+r}{1+i} \right)^{n-1}\\ & \mathrel{\stackrel{\scriptstyle i=r}{=}} 1 + \frac{1+i}{1+i} + \left(\frac{1+i}{1+i} \right)^2 + \ldots + \left(\frac{1+i}{1+i} \right)^n = 1+ \ldots + 1 = n \end{align*}\]

El mejor préstamos para Regina es una tasa de descuento del \(4.9\%\) para pagar la cantidad de \(\$5,501.62\) en exactamente \(4\) años. Ella puede cubrir \(\$5,000\) de su pago garantizado y añadir \(\$501.62\) extras al final de los \(4\) años. \[\begin{align*} &\underline{\text{Primero:}}\\ & \text{¿Cuál es la i que satisface:}\, 4500(1+i)^4 = 5000 \, \text{?}\\ &\Longrightarrow\: i = \left(\frac{10}{9} \right)^{1/4} = 0.02669 \: \Longrightarrow \: i = 2.2669\%\\ &\text{¿Cuál es la tasa de descuento equivalente} \, d^* \, \text{?}\\ &(1-d^*) = 1+i\\ &d^* = 1-V = 1 - \frac{1}{1+i} = \frac{i}{1+i} = \frac{0.02669}{1.02669} = 0.025\\ &\Longrightarrow\: \underline{d>d^*} \end{align*}\]
